# Welcome to Inertia

We want to build the best anti-cheat engine to help users across the board. We offer anti-cheat developers the best base they could have
Instead of rushing it, every time someone makes one, they should use us, then they could configure all the "fancy" stuff for themselves.
We don't want another war between what's best and who's not, as we're the users.
